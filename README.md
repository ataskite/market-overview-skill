# Daily Market Overview Skill

> 自动化生成每日市场概览笔记的 Claude Code 技能

[![Claude Code Skill](https://img.shields.io/badge/Claude_Code-Skill-blue)](https://claude.com/claude-code)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 📖 项目简介

这是一个为 [Claude Code](https://claude.com/claude-code) 设计的自动化技能，能够智能生成每日市场概览笔记并保存到 Obsidian vault。该技能通过并行爬取多个数据源，实时获取 A股、全球市场、大宗商品数据，智能筛选市场要闻，最终生成结构化的 Markdown 笔记。

### 核心功能

- 🕐 **智能时间识别**：自动判断交易日/假日模式，适配不同数据源
- 🚀 **并行数据获取**：多会话并行爬取，20秒内完成全部数据获取
- 📊 **全面市场覆盖**：A股指数、板块资金、全球市场、大宗商品、债券汇率
- 📰 **智能新闻筛选**：基于优先级的市场要闻筛选（国内+海外）
- 📝 **结构化输出**：生成符合 Obsidian 规范的 Markdown 笔记
- 🔄 **容错机制**：多级容错，部分数据缺失不影响整体生成

## 🏗️ 核心运行机制

### 工作流程概览

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  时间检查   │ →  │  数据获取   │ →  │  文件生成   │ →  │  输出确认   │
│交易日/假日  │    │  并行爬取   │    │  模板填充   │    │  清理会话   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

### Step 1: 时间与模式检查

首先获取当前时间并判断运行模式：

```bash
TZ='Asia/Shanghai' date '+%Y-%m-%d %H:%M:%S %Z %u'
```

**模式判断优先级**（从高到低）：

| 条件 | 模式 |
|:-----|:-----|
| 日期在节假日范围内 | 假日模式 |
| 日期在调休工作日列表中 | 交易日模式 |
| 周六/周日 | 假日模式 |
| 工作日 | 交易日模式 |

确定模式后加载对应模板：
- **交易日模式** → `references/template-trading-day.md`
- **假日模式** → `references/template-holiday.md`

### Step 2: 并行数据获取

使用 **playwright-cli** 多会话并行技术，同时发起 5 个浏览器会话：

```bash
# 5个会话并行启动（~8秒加载）
playwright-cli -s=a股   open "https://gu.sina.cn/m?type=hq#/index/index" &
playwright-cli -s=行情  open "https://www.cls.cn/quotation" &
playwright-cli -s=全球  open "https://gu.sina.cn/m?type=global#/world" &
playwright-cli -s=商品  open "https://wallstreetcn.com/markets/commodities" &
playwright-cli -s=收评  open "https://www.cls.cn/searchPage?keyword=收评&type=telegram" &
```

同时使用 **MCP 工具**并行获取新闻：

| MCP 工具 | 数据源 | 内容 |
|:---------|:-------|:-----|
| `mcp__fetcher__fetch_url` | 东方财富 | 国内要闻 |
| `mcp__local-web-reader__fetch_url` | 华尔街见闻 | 全球新闻 |

**总耗时**: 约 20-25 秒（并行执行）

### Step 3: 数据提取与处理

#### A股数据（交易日）

**数据源**: 新浪财经 + 财联社行情

| 数据项 | 说明 |
|:-------|:-----|
| A股指数 | 上证、深证、创业板、北证50、科创50 |
| 涨跌分布 | 上涨/平盘/下跌家数 |
| 涨跌中位数 | 市场真实温度 |
| 热门板块 | 前6个概念板块及领涨股 |
| 主力资金 | 净流入 TOP5 |

#### 全球数据（交易日+假日）

**数据源**: 新浪财经全球

| 类别 | 指标 |
|:-----|:-----|
| 全球指数 | 道琼斯、纳斯达克、标普500、恒生、日经225等10个 |
| 汇率 | 美元指数、离岸人民币、欧元/美元、英镑/美元 |
| 债券 | 中国10年期、美国10年期国债收益率 |

#### 大宗商品

**数据源**: 华尔街见闻

关注 6 个主要品种：现货黄金、现货白银、布伦特原油、WTI原油、伦敦期铜、伦敦期铝

#### 市场要闻

**数据源**: 东方财富 + 华尔街见闻

**筛选优先级**:
- 🔴 **最高**: 重大政策/监管、黑天鹅事件、美联储政策、AI突破
- 🟠 **高**: 市场异动、重要经济数据、美股七姐妹
- 🟡 **中**: 行业拐点、地缘事件、全球重要公司
- 🟢 **低**: 常规跟踪

**输出**: 国内10条 + 海外10条

### Step 4: 文件生成

#### 命名规则

```
06 市场观察/YYYY年市场记录/YYYY-MM-DD [核心内容15-30字].md
```

#### 涨跌配色

- 上涨: `<font color=#fb4b4b>+X.XX%</font>` (红色)
- 下跌: `<font color=#1fe059>-X.XX%</font>` (绿色)

#### 极简速览

用 200 字以内自然语言提炼当日要点，优先级：重大政策 > 市场异动 > 主线板块

### Step 5: 投研分析（可选）

**触发条件**（满足任一）:
- 用户明确要求
- 重大政策/黑天鹅事件
- 市场主线明显切换
- 成交量异常变化

**分析内容**: 搜索用户近1个月投研笔记，按关键词匹配，最多引用3篇

## 🚀 使用说明

### 安装

1. 确保已安装 [Claude Code](https://claude.com/claude-code)
2. 克隆本仓库到 Claude Code 技能目录：

```bash
cp -r daily-market-overview ~/.claude/skills/
```

3. 安装依赖工具：

```bash
# Playwright CLI (Claude Code 内置)
# MCP 工具 (在 Claude Code 配置文件中配置)
```

### 配置

在 `~/.claude.json` 中配置 MCP 工具：

```json
{
  "mcpServers": {
    "fetcher": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-fetch"]
    }
  }
}
```

### 触发词

- "市场怎么样"
- "今天市场"
- "市场表现"
- "收评速览"
- "行情怎么样"
- "市场概览"

### 输出位置

笔记默认保存到：`06 市场观察/YYYY年市场记录/`

## ✨ 特性与亮点

### 1. 智能容错

- **单源失败**: 标注"数据暂缺"，继续处理其他数据
- **全部失败**: 标注"⚠️ 数据源异常"
- **页面结构变化**: 用 AI 理解能力适应，不硬编码正则

### 2. 高效并行

- 5 个 Playwright 会话并行
- 2 个 MCP 工具并行
- 总耗时约 20-25 秒

### 3. 数据全面

| 类别 | 覆盖范围 |
|:-----|:---------|
| A股 | 5大指数 + 涨跌分布 + 板块资金 |
| 全球 | 10大指数 + 债券汇率 |
| 商品 | 6个主要品种 |
| 要闻 | 国内10条 + 海外10条 |

### 4. 智能筛选

新闻筛选遵循：主题多样性 > 时效优先 > 去噪 > 投资参考价值
