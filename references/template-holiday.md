# 周末/假日模式模板

> 此文件定义周末/假日市场概览的 Markdown 输出格式
>
> **📌 极简速览规则**：用200字以内自然语言提炼假日要点，优先级：重大政策/事件 > 全球市场异动 > 国内政策，先国内再全球。
>
> **⚠️ 智能投研分析**：仅在用户明确要求/重大政策/黑天鹅/主线切换时添加此部分，平时不要包含在笔记中。
>
> **⚠️ 假日数据限制**：假日模式下全球指数/商品/汇率/债券数据不自动获取。
> 仅当用户明确要求时，才单独获取全球资产数据（使用 playwright-cli 并行获取新浪财经全球页面 + 华尔街见闻商品页面），否则跳过全球资产部分。
