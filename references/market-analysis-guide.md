# 投研分析指南

## 触发条件

满足**任一**时执行：
- 用户明确要求
- 市场出现明显的**主线切换**信号
- **重大政策**或**黑天鹅事件**
- 市场成交量大幅变化

## 分析内容

```markdown
## 🧠 投研分析

### 📊 市场主线识别
**当前主线板块**：[今日涨幅最大的题材板块]
**主线强度**：[板块涨幅%] [持续性描述]
**主线阶段**：[启动期/加速期/高潮期/退潮期]

### 💡 历史策略验证
> **📋 参考**：[[笔记标题]]
[简明摘要，1-2句话]

### ⚠️ 策略矛盾识别（如有）
> 当前市场热点与历史策略存在冲突
> - 市场热点：[当前热点]
> - 历史观点：[历史策略]

### 🎯 配置建议
- **仓位建议**：[根据成交额和情绪]
- **板块配置**：[主线 + 辅助]
```

## 跳过条件

- 市场数据不完整
- 成交额过低（< 5000亿）
- 无明显主线（最大涨幅 < 3%）
- 用户未要求且无触发事件

## 笔记搜索

搜索近1个月用户投研笔记，按关键词匹配相关性，最多引用3篇。假日模式扩展到2个月。
